# Taskify // Modern Task Management Web Application

Taskify is a modern productivity-focused web application that helps users organize, manage, and track their daily tasks efficiently. The application provides a beautiful glassmorphic interface along with powerful task management features such as task creation, editing, completion tracking, filtering, sorting, theme switching, local storage persistence, and progress monitoring.

---

## 🎯 Objective

The primary goal of this project was to build a complete task management system using HTML, CSS, and Vanilla JavaScript while implementing modern UI/UX design principles and client-side data persistence.

Key project requirements included:

* **Task Management**: Create, edit, complete, and delete tasks.
* **Task Organization**: Categorize tasks and assign priority levels.
* **Dynamic Task Display**: Render tasks dynamically using JavaScript.
* **Search & Filtering**: Quickly find tasks using search and filter options.
* **Progress Tracking**: Display completion statistics and progress percentage.
* **Data Persistence**: Save tasks and theme preferences using LocalStorage.
* **Responsive Design**: Deliver a seamless experience across all devices.
* **Interactive User Experience**: Add animations, notifications, and celebrations.

---

## 🛠️ Tools Used

* **Frontend Structure**: HTML5 (Semantic Markup)
* **Styling & Design**: CSS3 (Flexbox, CSS Grid, Glassmorphism, Responsive Design)
* **Application Logic**: Vanilla JavaScript (ES6+, DOM Manipulation, LocalStorage API, Canvas API)

  * **Font Awesome** for icons.
  * **Google Fonts (Outfit & Inter)** for modern typography.

## 📝 Steps Performed

### 1. Project Planning & Feature Design

* Designed a productivity-focused dashboard layout.
* Defined task properties such as title, category, priority, due date, and completion status.
* Planned filtering, sorting, search, and progress tracking features.

### 2. User Interface Development (`index.html`)

* Created a responsive dashboard layout.
* Built sections for:

  * Progress Statistics
  * Task Creation Form
  * Search Functionality
  * Filter Controls
  * Sort Options
  * Dynamic Task List
  * Edit Task Modal
  * Notification System
* Integrated SVG illustrations and Font Awesome icons.

### 3. Styling & Responsive Design (`style.css`)

* Implemented a modern glassmorphism design system.
* Created Dark Mode and Light Mode themes.
* Added animated background glow effects.
* Designed responsive task cards, badges, buttons, and modals.
* Implemented hover effects, transitions, and smooth animations.
* Optimized layouts for desktop, tablet, and mobile devices.

### 4. Application Logic & Functionality (`app.js`)

* Developed complete CRUD functionality:

  * Add Tasks
  * Edit Tasks
  * Delete Tasks
  * Mark Tasks as Completed
* Implemented task filtering:

  * All Tasks
  * Active Tasks
  * Completed Tasks
* Added real-time task search functionality.
* Built sorting options:

  * Newest First
  * Oldest First
  * Due Date
  * Priority Level
* Stored task data and theme preferences using LocalStorage.
* Created a statistics engine to calculate progress and completion rates.
* Developed a custom toast notification system.
* Implemented a Canvas-based confetti celebration effect when all tasks are completed.
